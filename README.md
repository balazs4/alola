> converts `curl -isL` output (headers + body) into JSON

## Install

```bash
npx alola

# or

npm install -g alola
```

## Usage

```bash
curl -isL <url> | npx alola
```

### Example

```bash
curl -isL https://pokeapi.co/api/v2/region/alola | npx alola
```
