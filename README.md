> converts `curl -isL` output (headers + body) into JSON

## Install

```bash
npx alola

# or

npm install -g alola
```

## Usage

```bash
curl -isL <url> | npx alola
```

### Example

```bash
curl -isL https://swapi.co/api/people/1/ | npx alola
```
